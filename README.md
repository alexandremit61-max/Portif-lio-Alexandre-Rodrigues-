---

## 🛠️ Tecnologias Utilizadas
Para a construção deste projeto, utilizei as seguintes tecnologias:

- **HTML5:** Estrutura semântica para melhor acessibilidade e SEO.
- **CSS3:** Estilização moderna com **Flexbox** e **CSS Grid** para garantir responsividade em qualquer dispositivo.
- **JavaScript (ES6+):** Lógica para manipulação do Menu Mobile e efeitos de rolagem suave (Smooth Scroll).
- **FontAwesome:** Biblioteca de ícones profissionais.
- **Google Fonts:** Tipografia legível (Inter).
